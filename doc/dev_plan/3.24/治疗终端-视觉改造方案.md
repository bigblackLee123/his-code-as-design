## 七、数据来源说明

### 7.1 静区面板卡片数据映射

不新建表，在现有 `therapy_projects` 表上加 2 个可空字段：

| 新字段 | 类型 | 说明 |
|--------|------|------|
| `frequency` | VARCHAR(50) | 声波频率，如 `"7.83Hz"`、`"8-12Hz"`、`"432Hz"` |
| `frequency_band` | VARCHAR(100) | 频段描述，如 `"舒曼波/Alpha波段"`、`"Theta波"` |

**数据提取规则**（从现有 `mechanism` 字段提取）：

| 项目 | frequency | frequency_band | 提取来源 |
|------|-----------|---------------|---------|
| 舒曼波 | 7.83Hz | 舒曼波/Alpha波段 | "alpha波频率相似" |
| 温泉水疗音乐冥想 | 8-12Hz | Alpha波 | "alpha波使大脑快速放松" |
| 暖阳安驻 | 8-12Hz | Alpha波 | "叠加Alpha脑波" |
| alpha脑波音乐 | 8-12Hz | Alpha波 | "alpha波8-12Hz" |
| theta脑波音乐 | 4-8Hz | Theta波 | "theta波4-8Hz" |
| 432hz | 432Hz | 自然共振 | 项目名 "432hz" |
| 其余 30 项 | null | null | mechanism 中无频率信息 |

**卡片展示 fallback 逻辑：**

| demo 卡片位置 | 优先数据 | fallback | 示例 |
|--------------|---------|----------|------|
| 声波频光 - 主值 | `frequency` | `bpm` + " BPM" | `7.83 Hz` / `50 BPM` |
| 声波频光 - 副标签 | `frequency_band` | mechanism 截取前 15 字 | `Alpha波` / `自然情境深度减压` |
| 调式组织 - 主值 | `mood` | -- | `平静` / `积极、欢快` |
| 调式组织 - 副标签 | `energyLevel` | -- | `深度镇静` / `中度激活` |
