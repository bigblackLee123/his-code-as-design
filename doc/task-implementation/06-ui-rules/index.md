# HIS UI 生成规则（UI Generation Rules）

> AI 在生成任何 UI 代码时必须严格遵循以下规则。
> 技术栈：React + TypeScript + Tailwind CSS + shadcn/ui。

## 章节导航

| 章节 | 文档 | 说明 |
|------|------|------|
| 1 | [样式约束](./style-constraints.md) | 禁止内联样式、禁止自定义CSS、必须shadcn/ui、必须Tailwind |
| 2 | [代码结构约束](./code-structure.md) | 组件文件规范、导入规范、无障碍性要求 |
| 3 | [业务约束](./business-constraints.md) | 数据脱敏、数据密度 |
