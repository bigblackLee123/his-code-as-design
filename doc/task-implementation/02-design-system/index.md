# HIS 设计规范（Design System）

> 本文件是医院信息系统（HIS）的核心设计规范，定义了所有 Design Token（设计令牌），
> 作为 AI 生成 UI 代码的唯一设计来源。技术栈：React + TypeScript + Tailwind CSS + shadcn/ui。

| 2 | [排版规范](./typography.md) | 字体族、字号阶梯、行高比例 |

| 6 | [阴影](./shadows.md) | 阴影令牌定义 |
