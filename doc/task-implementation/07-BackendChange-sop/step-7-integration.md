## 步骤 7：前后端联调

前端对接后端接口，端到端流程验证。

- 确认前端调用方式（Supabase JS SDK / Edge Function invoke）
- 验证完整业务链路（用户操作 → 前端请求 → 后端处理 → 数据落库 → 前端展示）
- 检查错误处理（网络异常、数据校验失败、空数据场景）
- 如涉及前端 UI 变更，联调通过后转入 01-change-sop 步骤 5 继续

输出格式：
🔗 前后端联调

调用方式：xxx
业务链路验证：通过/失败
错误处理验证：通过/失败
需要转入前端 SOP：是/否
